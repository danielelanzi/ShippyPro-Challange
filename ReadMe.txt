Ho Sviluppato uno script in PHP che permette agli utenti di effettuare ricerche su un Database MySql con dei filtri che sono inseriti in degli input HTML in maniera dinamica, tramite i seguenti input:
    • Input Select per selezionare le città nel Database

L’applicazione è composta con i seguenti file:
    • index.php (Pagina di ricerca dei voli)
    • db_config.php (Pagina cconfigurazione per accesso al Database)
    • database.sql (Database completo di tutte le tabelle per la ricerca)
    • info.txt (Specifiche in merito alla versione usata di PHP, jQuery o di framework CSS impiegati nello sviluppo)
    • select_print.class.php (File di controller)

Lo script non è terminato nei seguenti punti:
